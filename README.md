# NxtBiz

NxtBiz is a spec-driven business operations automation platform. The codebase is split into:

- `client/` - Vite, React, Tailwind operations console
- `server/` - Express, MongoDB, Socket.IO, PDFKit API

## Phase 1 Scope

This scaffold establishes the project structure, visible NxtBiz branding, server health, authentication foundations, protected client routes, route/module placeholders, and model boundaries required by `spec.md`.

## Commands

```bash
npm install
npm --prefix server install
npm --prefix client install
npm run build
```

For local development:

```bash
npm run dev:server
npm run dev:client
```
