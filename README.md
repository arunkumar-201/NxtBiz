## 📌 Project Overview

NxtBiz is an AI-powered business operations platform designed to streamline customer management, email intelligence, workflow automation, ticket handling, reporting, invoice generation, and agent orchestration.

The platform combines modern full-stack development with asynchronous job processing using Redis and BullMQ to automate business operations efficiently.

## ⚡ Queue Architecture

Email Processing
→ Redis Queue
→ BullMQ
→ Worker Process
→ Agent Orchestration
→ Notifications

Features:
- Retry Attempts (3)
- Exponential Backoff
- Worker Concurrency (4)
- Automatic Cleanup
- Fallback Synchronous Execution
